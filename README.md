# React Spots



A modern React application that simulates a simple social media image-sharing platform. Users can update their profile information and create image posts through an interactive modal interface, with all changes reflected instantly in the UI using React state management.



## 🌐 Live Demo



https://react-spots.vercel.app/



## 📂 GitHub Repository



https://github.com/mavic01/react-spots



---



## Features



### Profile Management



- Edit profile information

- Update profile details dynamically

- Instant UI updates without page refresh



### Image Posts



- Add new image posts through a modal

- Upload an image URL

- Add image descriptions

- Newly created posts appear immediately in the gallery



### User Experience



- Responsive layout

- Reusable modal components

- Interactive user interface

- Dynamic rendering using React state



> **Note:** This project is frontend-only. Data is currently managed in application state and does not persist after a page refresh.

## Getting Started



### Clone the repository



```bash

git clone https://github.com/mavic01/react-spots.git

```



### Navigate into the project



```bash

cd react-spots

```



### Install dependencies



```bash

npm install

```



### Start the development server



```bash

npm run dev

```



---



## How It Works



1. View your profile information on the homepage.

2. Click **Edit Profile** to update your profile details.

3. Click **New Post** to open the post creation modal.

4. Enter an image URL and description.

5. Submit the form to instantly add the new post to the gallery.
